# Back-end Challenge 2021 🏅 - Space Flight News


## Sobre o projeto
RESTful API inspirada na Space Flight News API, que faz possível criar, editar, atualizar e excluir artigos que foram extraídos da Space Flight News API.

### Ferramentas

- FastAPI
